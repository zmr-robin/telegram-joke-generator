# telegram-joke-generator

This Telegram Bot with Python generates random jokes.
(My first Repositorie)

# Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```` 
pip install python-telegram-bot
pip install pyjokes
````
